# Working VWBridge script

The script that works in this VW/GBS environment is:

- `C:\Users\ammaganyane\tm\tm-context\src\vw-bridge\VWBridge.st`

## Why this one works

- It is written for **VisualWorks/Jade File In**
- It uses the correct VW/GBS namespace style such as `Root.Smalltalk` and `Core.*`
- It should be **Filed In**, not pasted into a workspace and executed with **Do It**

## How to use it

1. Open the VisualWorks/Jade **File Browser**
2. Select `VWBridge.st`
3. Choose **File In**

## Notes

- To stop the bridge in a workspace:
  - `VWBridge stop`
- To restart the bridge:
  - `VWBridge start`
